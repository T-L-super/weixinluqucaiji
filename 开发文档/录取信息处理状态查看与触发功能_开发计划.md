## 二、技术方案

### 2.1 后端技术方案

#### 数据库连接

```python
# MySQL 连接（源数据）
from sqlalchemy import create_engine
mysql_engine = create_engine('mysql+pymysql://user:pass@host/db')

# SQLite 连接（目标数据）
from sqlalchemy import create_engine
sqlite_engine = create_engine('sqlite:///path/to/admission_system.db')
```

#### API 设计

**1. 获取录取信息处理状态**

```
GET /api/admission/status
```

请求参数：
```json
{
  "start_time": "2026-04-01T00:00:00",
  "end_time": "2026-12-31T23:59:59",
  "keywords": ["录取", "offer"]
}
```

响应数据：
```json
{
  "data": [
    {
      "month": "2026-04",
      "status": "partial",  // all_processed, partial, none, no_data
      "total": 50,
      "processed": 30,
      "unprocessed": 20
    }
  ]
}
```

**2. 触发录取信息处理**

```
POST /api/admission/process
```

请求参数：
```json
{
  "month": "2026-04",  // 可选，不指定则处理全部
  "start_time": "2026-04-01T00:00:00",
  "end_time": "2026-04-30T23:59:59",
  "keywords": ["录取", "offer"]
}
```

### 2.2 前端技术方案

#### 组件结构

```
AdmissionStatusPanel.vue
├── 筛选区域（时间范围、关键词）
├── 月份状态表格
│   ├── 月份列
│   ├── 状态列（带颜色标签）
│   └── 操作列（查看、触发）
└── 触发确认对话框
```

#### 状态颜色定义

| 状态 | 颜色 | Element Tag Type |
|------|------|------------------|
| 已处理 (all_processed) | 绿色 | success |
| 部分处理 (partial) | 橙色 | warning |
| 未处理 (none) | 红色 | danger |
| 无数据 (no_data) | 灰色 | info |
